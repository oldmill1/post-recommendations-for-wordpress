Post Recommendations for WordPress

jQuery-powered recommendations.

How to use:

Put this code insidesingle.php, or anywhere you want. 

$args = array(); 
wp_recommendations($args);

Here are the options: 

$defaults = array( 
    'postID' => $post->ID,
    'size' => null,  
    'numberposts' => 6, 
    'orderby' => 'rand', 
    'show' => TRUE
);
